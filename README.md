# CSE 416 Midterm Cheat Sheet

A comprehensive, print-ready cheat sheet for CSE 416 (Machine Learning) midterm exam.

## 📄 Contents

- **Linear Regression** - fundamentals, MSE loss, gradient descent
- **Bias-Variance Tradeoff** - detection and fixes for under/overfitting
- **Regularization** - Ridge (L2) vs Lasso (L1) comparison
- **Cross-Validation** - model selection best practices
- **Classification & Logistic Regression** - decision boundaries, probability outputs
- **Fairness in ML** - WAE vs WYSIWYG worldviews
- **Data Preprocessing** - standardization rules, feature selection
- **Decision Trees** - complexity and regularization
- **Quick Reference** - formulas, algorithm decision guide, common pitfalls

## 🖨️ How to Use

1. Open `CSE416_cheat_sheet.html` in any web browser
2. Use **Ctrl+P** (Windows) or **Cmd+P** (Mac) to print
3. Select **Print to PDF** or **Print to Physical Printer**
4. Recommended: Print double-sided on 8.5×11 paper
